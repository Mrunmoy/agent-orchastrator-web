Quick start

from agent_orchestrator.storage import EventLogWriter, EventLogReader
from agent_orchestrator.storage.event_log import conversation_log_path

# Write events
log_path = conversation_log_path("conv-123")
with EventLogWriter(log_path) as writer:
    writer.append({
        "event_id": "evt-001",
        "event_type": "agent_message",
        "payload": {"text": "Hello from Codex"}
    })
    # timestamp is auto-added in UTC ISO-8601

# Read events
reader = EventLogReader(log_path)
all_events = reader.read_all()
recent = reader.tail(5)
since = reader.read_since("evt-001")

Key behaviours

  • Auto-timestamp: If the event dict has no timestamp key, one is injected as ISO-8601 UTC.
  • Atomic writes: Each append() call flushes to disk immediately. Data is readable without closing the writer.
  • Directory creation: Parent directories are created on writer init.
  • Missing file safety: Reader methods return empty lists if the JSONL file does not exist.
  • read_since(): Returns events after the matching event_id. Returns all events if the id is not found.
  • Path types: Both str and pathlib.Path accepted.

Gotcha: compact JSON separators

The writer uses separators=(",", ":") for compact output (no extra spaces). This keeps log files smaller but means manually reading them is slightly less pretty. Use python -m json.tool to pretty-print individual lines.
